# Sorting-Visualizer

This sorting visualizer is developed in python using tinkter library.

The sorting algorithms visualized are:
* Selection Sort
* Bubble Sort
* Merge Sort
* Quick Sort


## Technology Stack
* Python
* [tkinter](https://docs.python.org/3/library/tkinter.html)

## Local Installation

